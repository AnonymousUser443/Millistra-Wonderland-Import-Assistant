# 千星小助手 (Azure Inkstone Theme Design)

专为《原神》“千星奇域”模式设计的关卡导入与管理工具。

![Logo](https://github.com/AnonymousUser443/Millistra-Wonderland-Import-Assistant/blob/main/Resources/MainProgramLogo-256x256.png?raw=true)

## ✨ 功能特点

- **一键导入**：双击 `.gil` / `.gia` 文件即可自动导入到游戏目录。
- **多服支持**：自动识别国服 (CN) 和国际服 (Global) 路径，支持双修玩家。
- **智能图标**：为不同格式的文件提供精美的 "Azure Inkstone" 主题图标。
- **自动维护**：定期检查注册表关联，防止文件图标失效。

## 📥 下载与安装

请前往 [Releases 页面](这里填你的Release链接) 下载最新版本。

1. 下载 `千星奇域小助手_v1.0.zip`。
2. 解压到任意文件夹（**必须解压**，不能在压缩包里直接运行）。
3. 双击运行 `MillistraWonderlandImportAssistant.exe` 进行初始化。
4. 之后直接双击关卡文件即可使用。

## 🛠️ 开发环境

- **语言**: C# (.NET 8.0)
- **工具**: Visual Studio 2026
